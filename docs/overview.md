# Brotaria — Visão Geral

## O que é

Plataforma web de jardinagem virtual onde cada usuário possui um jardim pessoal composto por vasos e plantas únicas geradas por IA.

O foco principal é que cada planta possua uma identidade visual própria, persistente e evolutiva, baseada em um conjunto de características genéticas (DNA).

A evolução visual não deve gerar uma planta completamente diferente a cada estágio. A planta deve crescer mantendo sua identidade visual, adicionando novos elementos conforme evolui.

## Conceito Principal

Cada planta é composta por:
- DNA
- Estágio atual
- Histórico de evolução
- Imagem atual

**A imagem nunca é a fonte da verdade.**

A fonte da verdade sempre será o DNA da planta.

```
DNA
≠
Descrição Visual
≠
Imagem
```

---

## Objetivo Final

Criar um ecossistema onde cada planta possui:
- aparência única
- evolução consistente
- mutações raras
- identidade persistente

e onde a IA atua apenas como tradutora visual do DNA da planta, nunca como fonte da verdade dos dados.

Duas plantas com o mesmo DNA devem parecer parentes.

Duas plantas com DNA diferente devem parecer espécies diferentes.

Nenhuma planta deve perder identidade visual ao longo dos estágios.

A consistência visual é mais importante do que criatividade.
